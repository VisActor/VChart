# 辅助线的label如何自动换行？

## 问题描述
辅助线的label里面的文字如果太长的话可以自动换行吗？
[markline new line](/vchart/faq/14-0.png)

## 解决方案
你可以通过在`markLine.label.text`中设置数组来实现换行的效果。

## 代码示例

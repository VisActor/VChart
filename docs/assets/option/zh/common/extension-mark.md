#${prefix} extensionMark(Array)

extensionMark 是图表支持用户在图表系列上补充绘制任意内容的自定义接口。目前支持的图形类型如下

- `symbol` 点图形
- `rule` 一个线段，需要指定起点和终点
- `text` 文本
- `rect` 矩形
- `path` 路径
- `arc` 扇区
- `polygon` 多边形
- `image` 图片
- `group` 组，可以将其他 mark 放到组下

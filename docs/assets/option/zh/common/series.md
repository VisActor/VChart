{{ if: !${noInvalidType} }}

#${prefix} invalidType(string)

无效数据的处理方式配置。可选值为：

- `'break'`: 指在该数据点处断开
- `'link'`: 指忽略该点保持连续
- `'zero'`: 指该点默认数值为 0
- `'ignore'`: 指不处理

{{ /if }}

#${prefix} dataKey(string|string[]|function)

dataKey 用于绑定数据与 Mark 的关系，该配置在动画中非常重要。

该配置支持函数回调，函数的定义如下：

```ts
(data: Datum, index: number) => string;
```

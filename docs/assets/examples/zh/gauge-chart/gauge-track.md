# 带背景的仪表图

## 关键配置

- `categoryField`、`valueField` 属性分别用于指定数据类别与指针角度字段
- `innerRadius`、`outerRadius` 属性用于指定仪表盘的内外半径
- `startAngle`、`endAngle` 属性用于指定仪表盘的开始、结束角度
- `track` 属性可以配置 gauge 系列的背景

## 代码演示

```javascript livedemo
  data: [

      id: 'pointer',
      values: [

          type: 'A',

    {
      id: 'segment',
      values: [
        {
          type: 'Level 1',

          value: 9

          type: 'Level 3',

  series: [

      },
      track: {
        visible: true,
        style: {

      }
    },
    {
      type: 'gaugePointer',

      valueField: 'value',

      pin: {
        visible: true,
        width: 0.04,
        height: 0.04,
        isOnCenter: false,
        style: {
          fill: {
            type: 'threshold',
            field: 'value',
            domain: [6, 9],
            range: ['#E33232', '#FFC528', '#07A35A']
          }
        }
      },
      pinBackground: {
        visible: false
      },
      pointer: {
        width: 0.2,
        height: 0.1,
        isOnCenter: false,
        style: {
          fill: {
            type: 'threshold',
            field: 'value',
            domain: [6, 9],
            range: ['#E33232', '#FFC528', '#07A35A']

    {

      type: 'linear',

      outerRadius: 0.6,

      label: { visible: false }
    }
  ],
  indicator: [
    {
      visible: true,
      offsetY: '-10%',
      title: {
        style: {
          text: '76',
          fontSize: 80,
          fontWeight: 800
        }
      },
      content: [
        {
          style: {
            dy: 10,
            text: 'Current Score',
            fontSize: 24
          }
        }
      ]
    },
    {
      visible: true,
      offsetX: '-70%',
      offsetY: '45%',
      title: {
        style: {
          text: '0',
          fontSize: 18
        }
      }
    },
    {
      visible: true,
      offsetX: '70%',
      offsetY: '45%',
      title: {
        style: {
          text: '12',
          fontSize: 18
        }
      }
    }
  ]
};

const vchart = new VChart(spec, { dom: CONTAINER_ID });
vchart.renderAsync();

// Just for the convenience of console debugging, DO NOT COPY!
window['vchart'] = vchart;
```

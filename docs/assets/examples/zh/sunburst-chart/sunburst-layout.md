# 旭日图布局

旭日图支持配置对每一层进行布局控制

## 关键配置

- `innerRadius` 每一层的扇区内半径
- `outerRadius` 每一层的扇区外半径
- `gap` 每一层的扇区间隔
- `labelLayout` 标签布局, 对每一层的标签进行单独设置
  - `align` 对齐方式, `start` | `center` | `end`
  - `rotate` 标签的朝向 `tangential` | `radial`;
  - `offset` 偏移量;

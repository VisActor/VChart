# 坐标轴标签整体对齐

对于直角坐标系的坐标轴，我们可以通过在 `label` 上配置 `containerAlign` 属性来对所有的 label 进行整体的对齐方式调整。

## 关键配置

在 `axes` 属性上为指定方向的坐标轴配置 `label: { containerAlign: 'center' }` 可以让 label 整体居中展示

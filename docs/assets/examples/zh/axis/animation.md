# 坐标轴动画

坐标轴支持动画，但是默认是关闭的，如果需要开启，需要在 `axes` 属性上为指定的坐标轴开启动画属性 `animation`。

## 关键配置

在 `axes` 属性上为指定方向的坐标轴配置 `animation: true`

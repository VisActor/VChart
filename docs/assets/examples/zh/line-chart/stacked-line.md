# 堆叠折线图

堆叠折线图是折线图的一种，堆积折线图用于显示每一数值所占大小随时间或有序类别而变化的趋势。

这个例子对比了不同国家化妆品销量。通过堆叠折线图展示各个国家的不同类别化妆品销售情况，用于展示不同国家化妆品的销售趋势。

## 关键配置

- `seriesField` 属性用来声明参与颜色映射的字段
- `stack` 属性声明为 true 用来配置堆叠，会根据 `seriesField` 属性声明的字段进行堆叠

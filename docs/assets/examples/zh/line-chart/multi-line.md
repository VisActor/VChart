# 多系列折线图

多系列的折线图，且不进行堆叠。
这个例子展示了发达国家香烟消费的变化趋势。

> recreation of [https://www.datawrapper.de/charts](https://www.datawrapper.de/charts)。

## 关键配置

- `seriesField` 属性用来声明参与颜色映射的字段
- `legends` 属性用来配置图例

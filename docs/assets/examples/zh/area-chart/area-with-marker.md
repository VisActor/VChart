# 多系列面积图

面积图以图形方式显示定量数据。 它基于折线图。 通常用颜色，纹理和阴影线强调轴和线之间的区域。 通常，一个面积图会比较两个或多个数量。面积图适用于想要体现在连续自变量下，一组或多组数据的趋势变化以及相互之间的对比，同时也能够观察到数据总量的变化趋势。

## 关键配置

- `type: area` 属性声明为面积图
- `xField` 属性声明为分类字段或时序字段
- `yField` 属性声明为数值字段
- `stack`属性声明为是否堆叠
- `markLine` 属性声明为参考线，用于数据标记, `markLine.coordinates` 用于指定数据坐标

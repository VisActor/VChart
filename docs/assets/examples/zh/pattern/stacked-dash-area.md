# 末尾虚线堆叠面积图

当我们想对数据的某一段做一个区分的时候，可以设置线段的某一部分为特殊样式来突出这个特殊点。

## 关键配置

- `line` 图元的 `lineDash` 属性设置为回调函数，根据数据的 `forecast` 字段来判断是否为预测数据，如果是最新数据，则设置为虚线，否则设置为实线。
- `area` 图元的 `texture` 属性设置为回调函数，根据数据的 `forecast` 字段来判断是否为预测数据，如果是最新数据，则填充纹理。

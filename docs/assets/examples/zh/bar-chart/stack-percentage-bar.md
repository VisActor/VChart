# 百分比堆叠条形图

百分比堆叠条形图用于展示多个维度的数据占比情况。

## 关键配置

- `direction` 属性配置为 'horizontal'
- `xField` 属性声明为数值字段
- `yField` 属性声明为分类字段
- `stackField` 属性声明为堆叠字段
- `stack` 属性堆叠模式
- `percent` 属性配置为 `true` 开启数值百分比计算

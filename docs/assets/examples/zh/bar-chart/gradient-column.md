# 渐变圆角柱图

通过配置 bar 图元的样式，实现渐变色和圆角柱图。

## 关键配置

在 `bar.style` 属性中配置：

- `cornerRadius` 属性，设置圆角大小，当类型为数值时，表示四个角的圆角大小，当类型为数组时，表示左上、右上、右下、左下四个角的圆角大小。
- `fill` 属性中根据渐变色配置进行渐变色配置。

          type: 'sales'

        {

          y: 52,
          type: 'sales'
        },
        {
          x: 'sun',
          y: 48,
          type: 'sales'

        {

          y: 43,

        {

          y: 80,

          y: 72,
          type: 'profit'

        }

  ],

  yField: 'y',
  seriesField: 'type',
  bar: {
    style: {
      cornerRadius: 10,

        gradient: 'linear',

        y0: 0,
        x1: 0.5,
        y1: 1,
        stops: [
          {
            offset: 0,
            color: '#86DF6C'
          },
          {
            offset: 1,
            color: '#468DFF'
          }
        ]
      }
    },
    state: {
      selected: {
        stroke: '#000',
        lineWidth: 1
      }
    }
  },
  axes: [
    {
      orient: 'bottom',
      domainLine: {
        visible: false
      },
      bandPadding: 0,
      paddingInner: 0.1
    },
    {

        visible: false

        visible: true,

      },
      domainLine: {
        visible: false
      }

};

vchart.renderAsync();

# 百分比堆叠柱状图

通过配置 `stack` 属性和 `percent` 属性，实现百分比堆叠柱状图。

## 关键配置

- `seriesField` 属性用来声明参与颜色映射的字段
- `stack` 属性声明为 true 用来配置堆叠，会根据 `seriesField` 属性声明的字段进行堆叠
- `percent` 属性声明为 true 用来对数据进行百分比处理
- 在 `axes` 属性中为左轴的 label 进行了百分比的格式化（`formatMethod` 属性），让图表更易读

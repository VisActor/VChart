# 分组条形图

分组条形图用于展示多个分类的数据，通过条形的高度来对比不同分类的数据。

## 关键配置

- `direction` 属性配置为 'horizontal'
- `xField` 属性声明为数值字段
- `yField` 属性声明为分类字段，声明为数组类型，数组第二个值为分组字段
- `seriesField` 属性声明为颜色映射字段
- 通过为 `orient: left` 的坐标轴配置 `paddingInner` 属性为 0，调整组间间距

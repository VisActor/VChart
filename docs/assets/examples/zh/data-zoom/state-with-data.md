# 缩略轴以数据的方式定义初始范围

通常来说, 缩略轴的初始范围由 0 ～ 1 的比例数值表示. 在 VChart 中支持以数据的方式定义初始范围.

## 关键配置

- `startValue` 属性声明为初始范围左端点的数据
- `endValue` 属性声明为初始范围右端点的数据
- `start` 属性声明为初始范围左端点的比例, 范围 `[0, 1]`
- `end` 属性声明为初始范围右端点的比例, 范围 `[0, 1]`

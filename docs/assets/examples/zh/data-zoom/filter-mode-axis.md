# 轴缩放模式

通过直接改变轴的映射达到缩放轴的效果, 此时数据未被过滤.

## 关键配置

- `orient` 属性声明缩略轴组件相对图表的位置; `bottom`为图表底部, `top`为图表顶部, `left`为图表左侧, `right`为图表右侧
- `filterMode` 属性声明为缩略轴组件的缩放方式; `filter`为过滤数据从而达到缩放轴的效果, `axis`为直接缩放轴, 不过滤数据

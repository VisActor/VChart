# 形状词云图随机旋转角

文字的角度随机展示

## 关键配置

- `nameField` 属性声明为文字文本字段
- `valueField` 属性声明为文字大小字段
- `maskShape` 属性声明为形状轮廓
- `rotateAngles` 属性声明为核心词旋转角度的随机取范围, 即核心词旋转角度从该数组中随机选取
- `wordCloudShapeConfig.rotateAngles` 属性声明为填充词旋转角度的随机取范围, 即填充词旋转角度从该数组中随机选取

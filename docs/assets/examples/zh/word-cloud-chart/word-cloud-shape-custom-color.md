# 形状词云图自定义颜色

通过配置颜色列表自定义核心词和填充词的颜色映射

## 关键配置

- `nameField` 属性声明为文字文本字段
- `valueField` 属性声明为文字大小字段
- `maskShape` 属性声明为形状轮廓
- `colorList` 属性声明为核心词颜色随机选取范围, 即核心词颜色从该数组中随机选取
- `wordCloudShapeConfig.fillingColorList` 属性声明为填充词颜色随机选取范围, 即填充词颜色从该数组中随机选取

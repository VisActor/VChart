# 3D 面积图

在 2D 面积图的基础上配置 zField，即可被识别成 3D 面积图，然后配置 z 轴即可

## 关键配置

- `stack` 属性声明为 false 用来配置数据不采用堆叠，该属性在面积图中默认为 true，所以需要手动设置为 false

# Grid Line Fill

We can achieve a common zebra stripe effect using `grid.alternateColor`.

## Key Configuration

Configure the `grid.alternateColor` property on the `axes` property for the specified axis direction:

- If only one color fill is needed, use a string
- If multiple colors are needed for alternating fills, use a string array
